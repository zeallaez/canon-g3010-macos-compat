# Security policy

## Supported version

Security fixes are applied to the latest release.

## Reporting

Please do not include private documents, credentials, precise home-network
details, or full printer serial numbers in public issues.

For a suspected vulnerability, open a GitHub security advisory for this
repository when available. For ordinary compatibility bugs, use a public
issue with redacted diagnostics.

## Scope

The project configures local CUPS and DNS-SD services. It does not include the
Canon dependency and cannot provide security support for Canon software or
printer firmware.
